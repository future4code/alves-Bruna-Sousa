Sobre projeto Html e Css

Projeto Mine Lojinha

Hoje o objetivo é construir um sistema interno de uma loja usando somente HTML, sem CSS ou JavaScript. Vocês deverão escolher livremente os produtos e o nome da loja. Montamos uma estrutura prévia para a sua loja e vocês devem praticar algumas coisas que viram na aula de hoje

Todas as páginas do site devem passar por algum tipo de estilização
